#Фронтенд и бэкенд разработка - Контрольная работа №2

---

##Описание

В рамках данной контрольной работы было реализовано веб-приложение, дающее возможность управлять товарами и пользователями, с ситемой ролевого доступа (RBAC).

---

##Функционал

В данном приложении доступны следующие функции для каждой из нижеперечисленных ролей пользователей:

###Для гостя:
 - Вход в существующую учетную запись;
 - Регистрация с новыми учетными данными.

###Для пользователя:
 - Просмотр списка существующих товаров;
 - Просмотр данных своего профиля;
 - Выход из аккаунта.

###Для продавца:
 - Все возможности обычного пользователя;
 - Создание нового товара;
 - Редактирование или удаление существующего товара.

###Для администратора:
 - Все возможности обычного пользователя;
 - Просмотр списка зарегистрированных пользователей;
 - Редактирование данных любого из зарегистрированных пользователей;
 - Удаление аккаунта любого из зарегистрированных пользователей;

---

##Технологии

Данное приложение было реализовано с помощью следующих технологий:

 - React - для создания фронтенда;
 - Axios - для связи фронтенда и бэкенда через API;
 - SCSS Modules - для упрощения стилизации компонентов;
 - Node.js - среда выполнения для запуска сервера;
 - Express - фреймворк для создания серверной части;
 - JWT - для настройки авторизации и защиты маршрутов с помощью токенов доступа;
 - Bcrypt - для хэширования паролей;
 - Swagger - для создания документации API.
